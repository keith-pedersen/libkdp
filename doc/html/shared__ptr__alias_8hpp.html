Detailed Description

This file exposes the std::shared_ptr<U> alias constructor, for use by Cython implementation of Vector3, whose T() function creates a bound reference to the transverse vector (instead of a copy) by creating a shared_ptr bound to a Vector2*, but deleted as a Vector3*.
