Detailed Description

The base class for multi-dimensional histograms, with basic housekeeping, but without the actual histogram (avoid too much templating, because behavior changes too much between 1D and 2D implementations).

base ensures that the fileName is unique, so only one active histogram (per thread, not thread safe) is writing to a given file Inheritied classes only have to define Fill functions and Write
