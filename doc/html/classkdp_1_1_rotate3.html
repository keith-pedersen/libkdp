Detailed Description

template<typename real_t>
class kdp::Rotate3< real_t >

An active, Right-handed (RH) rotation of a 3-vector about an axis by an angle psi via Rodrigues' rotation formula.

Note
This class has been validated, and 99% of errors are smaller than O(1e-14). There is probably room for improvement, but we table this for now.

Construct the rotation that actively takes u to v, with a right-handed post-rotation of omega degrees about v.

The post-rotation is necessary to fully specify the rotation; "the rotation which takes u> to v>" is ambiguous because it only defines 2 of 3 Euler angles. There are actually an infinite number of rotations which take u> to v>, but only one for a given omega. For a detailed mathematical explanation, see Section A.3.1 in: "EXPANDING THE HEP FRONTIER WITH BOOSTED B-TAGS AND THE QCD POWER SPECTRUM"

Note
The identity operation only occurs when u || v and sin(omega) == 0
If v and u are antiparallel, then the rotation is ambiguous without supplying the axis; use the other constructor to define psi = Pi with a known axis.
Exceptions
Throwsinvalid_argument when v and u are antiparallel.
