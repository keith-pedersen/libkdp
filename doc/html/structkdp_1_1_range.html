Detailed Description

template<typename edge_t>
struct kdp::Range< edge_t >

A simple class for specifying a range (e.g. [10, 20]).

A useful feature is the ability to construct from an initalizer_list (i.e. Range x = {1, 2}). The ctor checks that min <= max and both min and max are numeric (finite or infinite, but NOT nan)
