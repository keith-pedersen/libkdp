Detailed Description

template<typename T>
class kdp::WelfordEstimate< T >

Use Welford's algorithm to calculate a running mean and variance.

This provides a numerically stable estimate of the variance, but without having to store the entire vector.

Note
There is no function for the standard deviation because some T may not fit into std::sqrt (e.g. a std::vector<double>).
