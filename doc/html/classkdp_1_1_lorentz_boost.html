Detailed Description

template<typename real_t>
class kdp::LorentzBoost< real_t >

An object that boosts along Axis() by boost factor Gamma() (the incoming axis is automatically normalized).

This is done using the boost equivalent of the Rodrigues formula. For a detailed mathematical explanation, see Section A.3.2 in: "EXPANDING THE HEP FRONTIER WITH BOOSTED B-TAGS AND THE QCD POWER SPECTRUM"
