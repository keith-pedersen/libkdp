Detailed Description

template<typename real_t>
struct kdp::Vector4< real_t >

Cartesian, Minkowski 4-vector.

Vector4 is defined in Minkowski space, not Euclidiean space To prevent metric confusion, the Mag and Dot functions are renamed versus Vector3: 

Vector3::Dot -> Vector4::Contract
Vector3::Mag -> Vector4::Length

There is also no Cross or InteriorAngle
